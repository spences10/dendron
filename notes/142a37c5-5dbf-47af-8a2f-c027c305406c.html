    Window management

    You can move the current window around the screen with a combination of the Windows key and the arrow keys.

    Holding down the Windows key+right with move the current window to the right hand half of the screen.

    To further reduce the size of the current window using the Windows key+down will move the window into the bottom right corner taking up a quarter of the screen.
