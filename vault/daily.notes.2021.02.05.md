## Notes

- To encode a string for use in a Twitter share intent use `escape`
  and the string you want to encode.

  - Example:

  ```js
  let str = `Here is one line
  and another`
  escape(str)
  // result
  // "Here%20is%20one%20line%0A%20%20and%20another"
  ```
