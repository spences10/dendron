## Notes

- To encode a string for use in a Twitter share intent use `encodeURI`
  and the string you want to encode.
  - Example:
    ```js
    let str = `Here is one line
    and another`
    encodeURI(str)
    // result
    // "Here%20is%20one%20line%0Aand%20another
    ```
